如果你需要通过URL访问，可以启动Vite开发服务器：

```bash
cd /Users/admin/Workspace/heartsphere_new/frontend
npm run dev
```

然后访问：`http://localhost:3000/temperature-demo.html`

**注意**：如果遇到端口占用错误，可以使用其他端口：


如果你有其他HTTP服务器工具，也可以使用：

## 🎨 演示页面功能

### 1. 角色展示区
- 150px的E-SOUL角色
- 可以切换8种表情（开心、难过、焦虑、平静等）
- 可以触发8种动作（挥手、点头、摇头、思考等）

### 2. UI组件展示
- **按钮样式**：主要、次要、文本三种类型
- **加载状态**：柔和的加载动画
- **Toast通知**：成功、错误、警告、信息四种类型

### 3. 智能问候
- 根据当前时间自动问候
- 早上/上午/中午/下午/晚上不同问候语

### 4. 交互反馈
- 点击表情按钮切换角色表情
- 点击动作按钮触发角色动作
- 所有操作都有Toast通知

## 🎯 快速开始集成

如果你想在自己的React项目中使用温度感组件：

### 1. 导入全局样式
```tsx
// 在你的入口文件（如index.tsx）中
import './tokens.css';
```

### 2. 使用UI组件
```tsx
import { Button, Input, Card } from './components/ui';

<Button variant="primary">主要按钮</Button>
<Input placeholder="输入..." />
<Card><p>卡片内容</p></Card>
```

### 3. 使用角色组件
```tsx
import { Character, useCharacterController } from './components/character';

const controller = useCharacterController();

<Character size={120} />
```

### 4. 使用对话服务
```tsx
import GreetingService from './services/GreetingService';

const greeting = GreetingService.selectGreeting({
  currentTime: new Date(),
  userEmotion: 'happy',
});
```

## 🔧 故障排除

### 问题1：打开HTML文件显示空白
**解决**：确保使用现代浏览器（Chrome、Firefox、Safari、Edge）

### 问题2：样式不生效
**解决**：检查tokens.css文件路径是否正确

### 问题3：端口占用
**解决**：使用不同端口启动
```bash
# 尝试不同端口
npx vite --port 5174
npx vite --port 5175
npx vite --port 5176
```
