## 🚀 立即开始测试

### 第一步：启动服务

**启动后端**:
```bash
cd backend
mvn spring-boot:run
```

**启动前端**:
```bash
cd frontend
npm run dev
```

### 第二步：打开测试文档

1. 打开 **开始测试.md** - 按照步骤快速开始
2. 或打开 **测试执行指南.md** - 查看完整流程
3. 使用 **前端测试快速清单.md** - 作为快速参考

### 第三步：执行测试

按照以下顺序执行：

1. **快速测试**（30分钟）
   - 应用初始化
   - 用户注册/登录
   - 创建世界/场景/角色
   - 基础对话功能

2. **详细测试**（按计划执行）
   - 参考"前端功能测试计划.md"
   - 参考"测试执行指南.md"

在测试过程中，使用以下格式记录：

```markdown
## 测试记录 - [日期]

### 测试模块: [模块名称]
**测试时间**: [开始时间] - [结束时间]
**测试人员**: [姓名]

#### 测试用例: [用例编号] - [用例名称]
- **状态**: ✅ 通过 / ❌ 失败 / ⚠️ 部分通过
- **实际结果**: [描述]
- **发现问题**: [如有问题，描述]
```

### 问题记录

使用问题记录模板记录所有发现的问题：

```markdown
## 问题记录

**问题编号**: BUG-XXX

**功能模块**: [模块名称]
**优先级**: P0/P1/P2
**状态**: 待修复/修复中/已修复
```
