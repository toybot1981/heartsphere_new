## 🚀 快速开始

### 1. 运行后端测试

```bash
cd backend

# 运行所有测试
./mvnw test

# 运行情绪系统相关测试
./mvnw test -Dtest=*Emotion*

# 运行记忆系统相关测试
./mvnw test -Dtest=*Memory*

# 运行集成测试
./mvnw test -Dtest=*Integration*
```

### 2. 运行前端演示

```bash
cd frontend

# 启动开发服务器
npm run dev

# 在浏览器中访问
# http://localhost:5173
```

### 3. 测试API

```bash
# 使用curl测试API
curl -X POST http://localhost:8080/api/emotions/analyze \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "text": "我今天非常开心！",
    "source": "conversation"
  }'
```
