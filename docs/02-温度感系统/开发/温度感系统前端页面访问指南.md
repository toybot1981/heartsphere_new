**完成日期**: 2025-12-28

---

## 🚀 启动前端服务

### 1. 进入前端目录

```bash
cd frontend
```

### 2. 安装依赖（如果还没安装）

```bash
npm install
```

### 3. 启动开发服务器

```bash
npm run dev
```

### 4. 访问页面

启动后，通常会在以下地址访问：
- **本地地址**: http://localhost:5173
- **网络地址**: 根据终端输出显示

## 🔧 集成演示组件

### 方式1: 在现有页面中添加

在需要展示演示的页面中导入：

```tsx
import { EmotionMemoryDemo } from './components/examples/EmotionMemoryDemo';

// 在组件中使用
<EmotionMemoryDemo userId={userProfile?.id || 0} />
```

### 方式2: 创建独立演示页面

创建一个新的路由页面：

```tsx
// src/pages/EmotionMemoryDemoPage.tsx
import { EmotionMemoryDemo } from '../components/examples/EmotionMemoryDemo';

export const EmotionMemoryDemoPage = () => {
  const userId = 1; // 从用户上下文获取
  
  return (
    <div className="container mx-auto p-4">
      <h1 className="text-2xl font-bold mb-4">情绪与记忆系统演示</h1>
      <EmotionMemoryDemo userId={userId} />
    </div>
  );
};
```

## 📝 注意事项

1. **后端服务**: 确保后端服务已启动（默认端口8080）
2. **用户认证**: 某些功能需要用户登录
3. **数据**: 首次使用可能没有数据，需要先进行对话
