# 前端服务启动指南

**完成日期**: 2025-12-28

---

## 🚀 启动前端服务

### 方式1: 使用npm（推荐）

```bash
cd frontend
npm run dev
```

### 方式2: 使用yarn（如果使用yarn）

```bash
cd frontend
yarn dev
```

---

## 📍 访问地址

启动成功后，在浏览器中访问：

- **本地地址**: http://localhost:3000
- **网络地址**: 根据终端输出显示（通常是 http://0.0.0.0:3000）

## 🔧 常见问题

### 端口被占用

如果3000端口被占用，可以：

1. **修改端口**（在vite.config.ts中）:
   ```typescript
   server: {
     port: 3001, // 或其他端口
   }
   ```

2. **或使用命令行指定端口**:
   ```bash
   npm run dev -- --port 3001
   ```

### 依赖未安装

如果遇到依赖错误：

```bash
cd frontend
npm install
```

---

## 📝 注意事项

1. **后端服务**: 确保后端服务已启动（默认端口8080）
2. **环境变量**: 检查`.env.local`文件配置
3. **首次使用**: 可能需要先登录或注册
