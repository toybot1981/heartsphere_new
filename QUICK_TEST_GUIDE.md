# 快速测试指南

## 运行测试

### 方式1: Python版本（推荐）

```bash
# 确保已安装requests库
pip3 install requests

# 运行测试（默认地址: http://localhost:8081）
python3 test-auth-init.py

# 或指定后端地址
python3 test-auth-init.py --base-url http://your-backend-url:8081
```

### 方式2: Bash版本

```bash
# 确保脚本有执行权限
chmod +x test-auth-init.sh

# 运行测试
./test-auth-init.sh

# 或指定后端地址
BASE_URL=http://your-backend-url:8081 ./test-auth-init.sh
```

## 测试内容概览

测试脚本会执行以下测试：

1. ✅ **服务可用性检查** - 验证后端服务是否运行
2. ✅ **系统配置检查** - 检查邀请码和邮箱验证码配置
3. ✅ **注册功能测试** - 包括正常注册、无效请求、重复检测、初始化验证
4. ✅ **登录功能测试** - 包括正常登录、错误密码、不存在用户
5. ✅ **用户信息获取** - Token验证、用户信息完整性
6. ✅ **初始化验证** - 世界创建、数据完整性
7. ✅ **首次登录初始化** - 新用户初始化流程

## 注意事项

- 测试会创建测试用户（用户名以`testuser_`开头），这些用户会保留在数据库中
- 确保后端服务已启动在指定端口（默认8081）
- 建议在测试环境运行，不要在生产环境使用
